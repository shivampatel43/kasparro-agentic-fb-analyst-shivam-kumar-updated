# Data

This folder contains a **small sample dataset** that is used for:

- local development
- unit tests
- fast, reproducible evaluation

The main config (`config/config.yaml`) points to `sample_fb_ads.csv` by default.

The columns follow the schema described in the assignment:

- campaign_name
- adset_name
- date
- spend
- impressions
- clicks
- ctr
- purchases
- revenue
- roas
- creative_type
- creative_message
- audience_type
- platform
- country
